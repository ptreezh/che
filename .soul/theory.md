# 理论框架详解

## 一、理论命题

### 命题1：认知多样性定理
在问题求解任务中，认知多样性群体优于同质高能力群体，当且仅当：
- 任务复杂度 C > 阈值 θ
- 认知方式互补性 Compl > 0.5
- 信息整合效率 η > 0.3

### 命题2：幻觉抑制定理
```
P_detect(hetero) = 1 - ∏ᵢ(1 - pᵢ) + S_interaction
```
S_interaction > 0 当且仅当认知异质性存在

### 命题3：进化稳定性定理
认知异质性维持需要：
- s > 1/(2N)
- μ ∈ [0.1, 0.4]
- f(x) 包含多样性奖励项

## 二、涌现机制

### 必要条件
1. H ≥ 0.5
2. 连通度 > 0.8
3. 有效整合机制

### 充分条件
1. 必要条件全部满足
2. Compl ≥ 0.6
3. S_interaction > 0

### 涌现量模型
```
CI = α × D_cognitive × I_synergy + β × C_complexity + ε
```
