# Nature/Science标准映射

## 一、期刊要求对比

| 维度 | Nature | Science | 本项目 |
|-----|--------|---------|-------|
| 原创性 | 首次发现 | 突破性 | ✅ |
| 重要性 | 重大影响 | 普适性 | ⚠️ 需强化 |
| 严谨性 | 最佳实践 | 稳健 | ✅ |
| 广泛兴趣 | 跨学科 | 多领域 | ✅ |

## 二、理论贡献层级

| 层级 | 描述 | 顶刊要求 | 本项目 |
|-----|-----|---------|-------|
| L3 | 提出机制 | 最低 | ✅ 已达成 |
| L4 | 建立原理 | 推荐 | 🎯 目标 |
| L5 | 形成范式 | 理想 | 长期目标 |

### 理论贡献评分
```
总分 = 0.15×概念清晰度 + 0.20×逻辑严谨性 + 0.25×预测准确性
     + 0.15×解释范围 + 0.10×理论简洁性 + 0.15×可证伪性

门槛：≥ 85/100
```

## 三、审稿问题应对

| 常见问题 | 准备材料 |
|---------|---------|
| "效应足够大？" | 效应量+置信区间分析 |
| "排除替代解释？" | 控制变量+混淆检验 |
| "可重复性？" | 复现实验数据 |
| "实际意义？" | 应用场景说明 |

## 四、投稿策略

### 首选期刊
1. Nature（跨领域影响力最大）
2. Science（方法创新认可度高）
3. Nature Machine Intelligence（AI领域顶刊）
4. PNAS（学院派认可）

### 投稿条件
```
□ 15代实验完成
□ 效应量 d ≥ 0.5
□ 复现通过
□ 专家预审通过
□ 质量对比达标
```
