# 检查清单

## 一、实验前检查

```
□ Python ≥ 3.9
□ Ollama运行正常
□ 模型已下载
□ 磁盘 ≥ 50GB
□ 内存 ≥ 16GB
□ 配置已更新
□ 随机种子已记录
□ 任务库已验证
□ 输出目录已创建
```

## 二、每日检查

```
□ 实验进度正常
□ 日志无严重错误
□ 数据完整性验证
□ 进度追踪更新
```

## 三、每代检查

```
□ 多样性 H > 0.5
□ 性能趋势正常
□ 检查点已保存
□ 无异常数据点
```

## 四、实验后检查

```
□ 15代全部完成
□ 数据完整性通过
□ 效应量 d ≥ 0.5
□ p < 0.01
□ 复现实验成功
```

## 五、论文检查

```
□ 字数符合限制
□ 所有声明有数据支持
□ 图表 ≥ 300 DPI
□ 色彩色盲友好
□ 误差棒已标明类型
□ 参考文献格式正确
```

## 六、投稿前检查

```
□ 论文结构符合期刊
□ 补充材料完整
□ 作者贡献声明
□ 利益冲突声明
□ 数据可用性声明
□ 100%清单通过
```
