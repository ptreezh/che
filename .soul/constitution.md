## 一、统计验证规范

### 1.1 样本量计算
```python
# 先验功效分析
n = 2 × [(Z_(1-α/2) + Z_(1-β)) / d]²

# 本项目：d=0.7, α=0.05, power=0.85
n_per_group ≈ 36
n_total = 16,200 >> 36  # 远超所需
```

### 1.2 多重比较校正
| 比较次数 | 方法 |
|---------|-----|
| k ≤ 5 | Bonferroni |
| 5 < k ≤ 20 | Holm-Bonferroni |
| k > 20 | FDR |

### 1.3 效应量报告
| 检验 | 指标 | 阈值(小/中/大) |
|-----|-----|---------------|
| t检验 | Cohen's d | 0.2/0.5/0.8 |
| ANOVA | η² | 0.01/0.06/0.14 |
| 相关 | r | 0.1/0.3/0.5 |

## 二、验证体系

### Layer 1: 内部一致性
| 检验 | 标准 | 时机 |
|-----|-----|-----|
| Cronbach's α | ≥ 0.70 | 每代后 |
| ICC | ≥ 0.80 | 每代后 |
| KR-20 | ≥ 0.70 | 二分变量 |

### Layer 2: 交叉验证
```
K-fold CV (K=5-10)
报告：CV准确率 = mean ± std
```

### Layer 3: 独立复现
- 不同随机种子 ≥ 3次
- 效应量变化 < 20%
- p值均 < 0.05
