## 二、不可违背规则

### 2.1 零容忍模糊

| 禁止 | 替代 |
|-----|-----|
| "可能/或许/似乎" | 概率值 P = 0.xx |
| "显著提升" | d = x.xx, p < 0.xx |
| "效果很好" | 效应量 + 置信区间 |

**强制格式**：`[效应] t(df)=x, p<0.xx, d=x.xx (95% CI [x,y]), power=x.xx`

### 2.2 可证伪性

每个假设必须定义证伪条件：

| 假设类型 | 证伪条件 |
|---------|---------|
| 相关性 | r < 0.3 或 p > 0.05 |
| 因果性 | β < 0.1 或 p > 0.05 |
| 组间差异 | d < 0.2 |

### 2.3 透明度

- 所有数据/代码/参数公开
- 负面结果必须报告
- 边界条件必须声明

## 四、核心概念定义

### 4.1 认知异质性 H

```
H = -Σ(nᵢ/N) × log(nᵢ/N)

量化标准：
- 低：H < 0.3
- 中：0.3 ≤ H < 0.6
- 高：H ≥ 0.6
```

### 4.2 集体智能 CI

```
CI = P_group - P_expected

判定：
- CI > 0, p < 0.05 → 正向涌现
- |CI| < 0.05 → 无涌现
- CI < 0, p < 0.05 → 负向涌现
```

### 4.3 幻觉抑制 HS

```
HS = (D_true × 0.5 + D_false × 0.5)

评分标准(0-2)：
- 0.0：盲目接受
- 1.0：识别问题
- 2.0：明确拒绝
```

## 七、验证三层次

```
Layer 1: 内部一致性
├── Cronbach's α ≥ 0.70
├── ICC ≥ 0.80
└── 每代结束后执行

Layer 2: 交叉验证
├── K-fold CV (K≥5)
├── 报告 mean ± std
└── 分析批次完成后执行

Layer 3: 独立复现
├── 不同随机种子 ≥ 3次
├── 效应量变化 < 20%
└── 最终结果前执行
```

## 八、决策框架

### 8.1 实验决策

```
效应显著？
├── 是 → 通过验证层 → 继续下一阶段
└── 否 → 诊断原因
    ├── 功效不足 → 扩大样本
    ├── 效应不存在 → 修改假设/报告负面结果
    └── 方法问题 → 改进设计
```

### 8.2 写作决策

```
每个陈述是否有数据支持？
├── 是 → 保留
└── 否 → 删除或补充实验
```
