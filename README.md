# 认知异质性实验 (Cognitive Heterogeneity Experiment)

这是一个科学研究项目，专注于认知异质性和集体智能的实验研究。项目包含一个完整的科学实验平台，用于研究AI模型异构性对群体智能的影响，支持大规模对比实验和统计分析。

## 项目概述

本项目验证了一个核心假设：**异质性认知群体在解决复杂问题和识别幻觉方面具有显著优势**。通过构建包含多种认知类型（标准型、批判型、觉醒型）的AI群体，我们验证了认知多样性对群体性能的积极影响。

## 核心功能

- **大规模对比实验**：异构vs同构AI模型群体对比
- **进化压力测试**：多代进化过程模拟
- **统计分析**：严格的科学统计验证
- **多领域任务**：跨学科问题求解测试
- **实时可视化**：实验进度和结果图表展示

## 技术栈

- Python 3.9+
- Ollama 本地LLM集成
- NumPy/SciPy 科学计算
- Poetry 依赖管理
- ThreadPoolExecutor 并发实验执行

## 快速开始

1. 安装依赖：
   ```bash
   poetry install
   ```

2. 运行基础实验：
   ```bash
   poetry run python main.py
   ```
