Project Overview

At AI Psy Lab, we are dedicated to exploring the deep cognitive characteristics of AI systems. The Cognitive Heterogeneity Experiment aims to reveal how AI agents with different cognitive patterns collaborate to generate swarm intelligence that surpasses individual agents.

认知

Cognitive Heterogeneity

AI agents with different cognitive patterns work together to generate swarm intelligence that surpasses individual agents. This diversity is similar to the combination of members with different professional backgrounds in human teams.

科学

Scientific Validation

Based on psychological theory frameworks, we validate the positive impact of cognitive heterogeneity on group performance through strictly controlled experiments, ensuring the scientific nature and reproducibility of the results.

创新

Technical Innovation

Implementing the CloudAgent architecture, supporting multiple cloud service providers, breaking free from hardware dependencies, and flexibly invoking different models for experiments.

Core Concepts

多样性

Cognitive Diversity

Cognitive heterogeneity refers to a group of agents where different members have different cognitive patterns, thinking frameworks, and problem-solving strategies. This diversity can generate more comprehensive and innovative solutions.

假设

Experimental Hypothesis

Heterogeneous AI groups perform better than homogeneous groups in hallucination detection tasks. Cognitive diversity is positively correlated with group problem-solving ability, and performance is continuously optimized through evolutionary mechanisms.

性能

Performance Enhancement

Experimental results verify the core hypothesis: heterogeneous groups significantly outperform homogeneous groups in hallucination recognition, and the cognitive diversity index is strongly positively correlated with group performance.

Experimental Design

智能体

Agent Types

  • • Standard Agent: Follows instructions directly to provide comprehensive answers
  • • Critical Agent: Carefully analyzes factual accuracy, questions false premises
  • • Awakened Agent: Continuously reflects and questions established "common sense", challenging system biases
流程

Experimental Process

  • • Initialization: Build heterogeneous group
  • • Task Execution: Group responds to tasks with false premises
  • • Evaluation: AI evaluator scores
  • • Evolution: Remove low scores, replicate high scores
  • • Iteration: Repeat execution across multiple generations
技术

Technical Innovation

  • • CloudAgent Architecture: Supports multi-cloud services
  • • Real-time Monitoring: Performance visualization
  • • WebUI Interface: Intuitive operations
  • • Data Export: Complete reports
